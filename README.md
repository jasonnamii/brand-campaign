# brand-campaign

> 🇰🇷 한국어 README: [README.ko.md](README.ko.md)

Brand Campaign Engine — 5 Layers × 7 Domains × 3 Modes. Binet & Field 60:40 dual-track · Byron Sharp DBA · Korea default True.

## Overview

A Claude skill for designing, diagnosing, and reviewing brand campaigns. Built on IPA Databank (996 cases, Binet & Field), Byron Sharp's "How Brands Grow", Ogilvy Big Idea, Effie 4-pillar, and Google ABCD framework. Korean market overlay applied by default.

## Structure

- **5 Layers**: L1 Strategy · L2 Creative · L3 Media · L4 Execution · L5 Measurement
- **7 Domains**: D1 Launch · D2 Repositioning · D3 Renewal · D4 Sales Push · D5 Experience/Pop-up · D6 CSR/ESG · D7 Crisis
- **3 Modes**: M1 Diagnosis · M2 Design · M3 Retrospective

## 7 Absolute Rules

1. 60:40 Brand:Activation dual-track mandatory
2. Channel-Creative Fit enforced
3. DBA (Distinctive Brand Assets) preservation
4. Korea default True (offshore explicit opt-out)
5. 5 Layers independently callable
6. PREFLIGHT 4-check
7. Measurement 3-axis separation (Brand · Activation · Long-term)

## Redflag Catalog (10)

뒷광고 non-disclosure · Brand<60% · Channel all-in · DBA replacement · ESOV negative · Last-click only · Mega-only influencer · K-IP over-dependence · Brand safety gap · Brand Track absent.
