## 필수 산출

| 항목 | 내용 | 출처 |
|------|------|------|
| Big Idea Statement | 1문장·10년 유지 가능·채널·카피 무관 확장성 | Ogilvy·Bernbach |
| Key Message | 단일 핵심 메시지 (Rosser Reeves USP) + 증거 | Reality in Advertising |
| DBA (Distinctive Brand Assets) | 색상·로고·폰트·사운드·캐릭터·슬로건 — Fame·Uniqueness 점수 | Byron Sharp |
| 채널별 Adaptation Key | 플랫폼별 5-15초/15-30초/30-60초 스펙 + Hook(3초)·CTA | Channel-Fit |
| Tone & Voice | 브랜드 보이스 — copywriting-engine 연계 | - |
| 크리에이티브 영역 | Emotional vs Rational 비율, IPA 데이터상 Emotional 우위 | Binet&Field |

## DBA 보전 원칙 (절대규칙 3)

| 원칙 | 적용 |
|------|------|
| 색상·로고·사운드·캐릭터는 10년+ 유지 | 리뉴얼(D3) 시 교체 금지 |
| DBA Audit 주기적 수행 | Fame(인지도) × Uniqueness(독점성) 2×2 매트릭스 |
| Low Fame + Low Uniqueness | 폐기·재설계 |
| High Fame + Low Uniqueness | 교체 고려 (경쟁 오인) |
| Low Fame + High Uniqueness | 노출 강화 |
| **High Fame + High Uniqueness** | **절대 교체 금지** |

## Emotional vs Rational

- Binet&Field: Emotional 캠페인이 Rational 대비 장기효과 2배
- Rational 단독 캠페인 = 단기 sales lift만, 브랜드 자산 축적 ✗
- B2B도 Emotional 포함 권장 (CEB·LinkedIn 연구)

## Gotchas
- "완전히 새로운" 크리에이티브 요구 → DBA 교체 유도 함정
- Big Idea ≠ 단순 슬로건/태그라인. Big Idea는 10년 포괄, 태그라인은 1-3년
- 채널 무시 단일 크리에이티브 = 절대규칙 2 위반
- Emotional을 "감성팔이"로 오해 → Fame·Meaningful·Different 3축(Kantar) 필요
