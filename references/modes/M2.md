# M2 — 설계 (신규 캠페인 설계)

**입력:** 목표·타깃·예산·도메인
**출력:** 캠페인브리프.md (Brief → Big Idea → Media Plan → Measurement Plan 일체)

## 설계 흐름

1. PREFLIGHT 4체크 (모드 M2 확정)
2. **L1 STRATEGY** — Brief 작성
3. **L2 CREATIVE** — Big Idea + DBA + Adaptation Key
4. **L3 MEDIA** — Channel-Fit + 60:40 + 플라이트
5. **L4 EXECUTION** — 캘린더 + 스펙 + 컴플라이언스
6. **L5 MEASUREMENT** — Plan 사전 확정 (사후 측정 금지)
7. 도메인 오버레이 (D1~D7)
8. 한국 오버레이
