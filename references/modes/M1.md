# M1 — 진단 (기존 캠페인 리뷰)

**입력:** 기존 캠페인·브리프·실행 자료
**출력:** 진단메모.md

## 진단 흐름

1. PREFLIGHT 4체크 — 모드(M1 확정)·도메인 추정·예산·타깃
2. **Redflag 10종 스크린** (1차 필터) — Red/Yellow/Green 태깅
3. **5층 건강도** — L1~L5 각각 진단
4. **리프트 추정** — 가용 데이터 기반
5. **처방** — 우선순위 3-5개

## 출력 구조

```markdown
# [캠페인명] 진단메모

## 0. PREFLIGHT
- 모드/도메인/예산/타깃

## 1. Redflag 스크린
| # | Redflag | 판정 | 증거 |

## 2. 5층 건강도
| 층 | 점수(0-5) | 강점 | 약점 | 처방 |
| L1 STRATEGY | | | | |
| L2 CREATIVE | | | | |
| L3 MEDIA | | | | |
| L4 EXECUTION | | | | |
| L5 MEASUREMENT | | | | |

## 3. 리프트 추정
- Brand Lift: [%] (DEEP/LIGHT 표기)
- Sales Activation: [ROAS/CTR/CPA]
- Long-term: [NPS·재구매]

## 4. 처방 (우선순위)
1. [최우선·즉시]
2. [중요·30일]
3. [개선·90일]

## 5. 한국 오버레이 점검
- 네이버·카카오 채널 / 뒷광고 / K-IP / 팝업

## 6. 변호사·전문가 상담 권고
- 필요 영역: [contract / legal / privacy 등]
```

## 진단 점수 기준

- 5: 업계 최고 수준
- 4: 평균 이상
- 3: 평균
- 2: 평균 이하
- 1: 위험·재설계 필요
- 0: 미수행·없음

## 처방 우선순위 기준

- 절대규칙 위반 = 최우선 (예: 60:40 미달, DBA 교체)
- Red Redflag = 최우선
- Yellow Redflag = 30일 내
- 5층 건강도 점수 ≤2 = 60-90일 내
