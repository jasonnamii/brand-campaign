## 크리에이티브 원칙 (채널별)

### TikTok
- Native Storytelling (광고 같지 않은 광고)
- Sound-on (소리 켜진 시청 90%)
- Hook 3초·Pattern Interrupt
- 트렌드·해시태그 챌린지 활용
- Spark Ads로 organic 후 부스트

### IG Reels
- 정제된 비주얼 (TikTok 대비 high production)
- 전환 의도 강조 (CTA 명확)
- Story·Feed 연계 (Cross-format)
- Shopping 통합 (Reels Ads)

### YouTube Shorts
- Discovery 중심 (구독자 외 노출)
- 중립 톤 (브랜드 강조 ✗)
- Long-form YT 채널과 연계

### YouTube In-stream (ABCD 프레임 - Google)
- **A**ttract — 첫 5초 강력
- **B**rand — 브랜드 노출 5초 내
- **C**onnect — 감정·스토리·메시지
- **D**irect — 명확 CTA
- Skippable: 15-30s / Bumper: 6s / Non-skippable: 15s 제한

### 네이버
- 검색 의도 기반 (의도 명확)
- 브랜드 검색 광고 (자사 브랜드 검색 시)
- 블로그 마케팅 (체험단·SEO)
- 쇼핑 광고·플레이스 (오프라인)

### 카카오톡
- 비즈보드 (1:1 카드형) — 7자 카피 필수
- 채널 메시지 (구독자 푸시) — Open 3-5배
- 모먼트 (퍼포먼스 광고)
- 알림톡 (트랜잭션·CRM)

### 인플루언서
- Micro/Nano 우선 (전환·신뢰)
- 1회성 ✗ → 6개월+ 시리즈 (신뢰 축적)
- AIDA 메시지 흐름 (Attention·Interest·Desire·Action)
- 뒷광고 표시 의무 (절대규칙 4)

### OOH·DOOH
- 물리적 Availability·Fame 부스트
- 단순 메시지 (3초 인지)
- DOOH는 시간·날씨·이벤트 트리거
- 강남·홍대·여의도·성수 핵심 입지

## Gotchas

- 단일 크리에이티브 단순 리사이즈 = Channel-Fit 위반 (절대규칙 2)
- TikTok에 정제된 광고 톤 = 알고리즘 거부
- IG Reels에 TikTok 그대로 = 전환 미흡
- YT In-stream ABCD 무시 = Brand Lift 미달
- 네이버 검색 광고 무시 = 한국 검색자 63% 포기
- Kakao 비즈보드 7자 초과 = 가시성 급락
