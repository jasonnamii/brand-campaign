# Big Idea 검증 (Ogilvy 5 Questions)

David Ogilvy의 Big Idea 검증 5질문. 모두 YES = 진짜 Big Idea.

## 5 Questions

1. **Did it make me gasp when I first saw it?** — 첫 노출에서 숨이 멎었는가?
2. **Do I wish I had thought of it myself?** — "왜 내가 생각 못 했지?" 후회되는가?
3. **Is it unique?** — 카테고리·경쟁 대비 독창적인가?
4. **Does it fit the strategy to perfection?** — 전략과 완벽히 부합하는가?
5. **Could it be used for 30 years?** — 30년 사용 가능한가? (즉, 트렌드 종속 ✗)

## 추가 검증 (현대 보강)

| 질문 | 의미 |
|------|------|
| 6. Channel-agnostic? | TV·디지털·OOH·체험 모두 적용 가능한가 |
| 7. DBA 강화 가능? | Big Idea가 기존 DBA를 더 각인시키는가 |
| 8. Light Buyer에게도 닿는가? | Heavy User만 타깃 ≠ Big Idea |
| 9. 측정 가능한 효과? | Brand Lift·Mental Availability 향상 가능 |
| 10. 윤리·문화 충돌 없음? | 다양성·포용·세대 충돌 점검 |

## Big Idea vs 슬로건/태그라인 구분

| 구분 | Big Idea | 슬로건·태그라인 |
|------|----------|---------------|
| 수명 | 10-30년 | 1-3년 |
| 범위 | 전체 캠페인·여러 채널·여러 실행 | 단일 카피 |
| 깊이 | 브랜드 본질 | 표면 메시지 |
| 예시 | Apple "Think Different" 계열 정신 | "Get a Mac" 캠페인 슬로건 |

## Big Idea 도출 워크숍 프로세스

1. **Brief 깊이 읽기** — L1 STRATEGY 출력 검토
2. **Insight Mining** — 소비자·카테고리·문화 3축
3. **Tension 발견** — 갈등·불편·열망
4. **여러 안 발산** — 30-50개 (양 우선)
5. **5 Questions 검증** — Ogilvy 필터
6. **DBA 호환 검증** — 기존 자산 강화 가능?
7. **Channel Test** — 5개 채널 적용 시뮬

## Gotchas

- 트렌드·밈 기반 = Big Idea 아님 (수명 부족)
- "Just Different"만 추구 = Strategy 미연결
- Heavy User 인사이트만 사용 = Light Buyer 누락
- 단일 카피·단일 비주얼 = 슬로건이지 Big Idea ✗
- 첫 안에 안주 = Ogilvy "30-50 발산" 원칙 위반
