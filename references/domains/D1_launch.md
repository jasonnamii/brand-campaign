# D1 — 런칭 (신제품·신브랜드)

**핵심 목적:** Mental Availability 구축 — 카테고리 CEPs와 브랜드 연결

## 특이 전략

| 항목 | 설정 |
|------|------|
| 60:40 조정 | **80:20 (첫 6개월)** — 브랜드 빌딩 집중 |
| Reach 우선 | 타깃 카테고리 사용자 60%+ 도달 |
| DBA 확립 | 첫 캠페인에서 색상·로고·사운드·캐릭터 동시 각인 |
| CEPs | 5-10개 CEP 동시 확보 시도 (단일 CEP 리스크) |
| 플라이트 | Burst 6주 + Continuous 6개월 |

## 채널 우선순위

1. YouTube In-stream (Brand Lift·ABCD 프레임)
2. TikTok (공유·Fame)
3. OOH·DOOH (물리적 Availability)
4. 인플루언서 Mega+Micro 혼합
5. 네이버·카카오 (Activation 보조)

## KPI (런칭 6개월)

| 축 | 지표 | 목표 |
|---|------|------|
| Brand | Aided Awareness | Δ +15pt |
| Brand | Unaided Awareness | 5%+ |
| Brand | Mental Availability (CEPs) | 3-5개 연결 |
| Activation | 검색량 | 3배+ |
| Activation | 첫 구매 전환 | baseline 설정 |
| Long-term | NPS baseline 측정 | 조사 실시 |

## 한국 특이

- 팝업 런칭 이벤트 (D5 연계) — 성수동·연남
- K-IP 콜라보 (뉴진스·아이브·Stray Kids 등) — 첫 Reach 부스트
- 네이버 쇼핑·브랜드 검색 광고 필수 (검색자 63%)

## Redflag

- Heavy User만 타깃 (Byron Sharp: Light Buyer·비고객 침투가 성장)
- 런칭 후 Activation 매출에만 매달림 → Mental Availability 미구축
- DBA 여러 버전 시도 (실험) → 각인 실패
