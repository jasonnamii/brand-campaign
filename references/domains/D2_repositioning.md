# D2 — 리포지셔닝 (브랜드 재정의·카테고리 확장)

**핵심 목적:** 기존 브랜드를 새 시장·새 타깃·새 카테고리에 연결. **기존 DBA 유지·신규 CEPs 확보**

## 특이 전략

| 항목 | 설정 |
|------|------|
| 60:40 | 60:40 기본 |
| DBA 원칙 | **절대 교체 금지** — 색상·로고·사운드 유지 |
| POP·POD | 새 카테고리 POP(Parity) 먼저 확보 → POD(Difference) |
| 플라이트 | Pulsing 12개월 — 재인식 반복 |
| Long-term KPI | 카테고리 인식 전환 12-24개월 |

## 채널 우선순위

1. YouTube 스토리텔링 (30-60s) — 변화 맥락 설명
2. 네이티브·PR — 신뢰·정당성
3. 인플루언서 Macro·Micro — 새 카테고리 전문가
4. OOH — 물리적 재각인
5. 검색 광고 (네이버) — 새 키워드 점유

## 사례 패턴

- **Volvo: 안전 → 프리미엄 라이프스타일** — 기존 안전 유산 유지+감성 확장
- **올드 스파이스: 중장년 → 20-30대** — Manhood 재정의, DBA(휘파람 소리) 유지
- **IBM: 하드웨어 → 솔루션** — Blue DBA 유지, 카테고리 전환

## KPI

| 축 | 지표 | 목표 |
|---|------|------|
| Brand | Brand Attribute Shift | 핵심 속성 Δ +10pt |
| Brand | CEPs 확장 | 신규 CEP 3-5개 연결 |
| Brand | Mental Availability | 신 카테고리 Top 5 진입 |
| Activation | 신 카테고리 전환 | 전년 동기 대비 +30% |
| Long-term | 프리미엄 가격 유지 | 기존 가격대 유지 가능 |

## Redflag

- DBA 교체로 "재출발" 시도 → Redflag 4 (기존 자산 파괴)
- POP 무시하고 POD만 강조 → 카테고리 불인정
- 12개월 미만 단기 리포지셔닝 → 인식 전환 실패
- 기존 Heavy User 이탈 관리 부재

## Gotchas
- "완전히 새로운 브랜드로" = 리포지셔닝 아님 → D1 런칭이 맞음
- 가격 포지셔닝 변경은 가장 어려움 (상향 조심·하향 쉬움)
- 리포지셔닝 실패 사례: GAP 로고 변경(2010) — 1주일 롤백
