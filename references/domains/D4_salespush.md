# D4 — 세일즈푸시 (프로모션·시즌·이벤트)

**핵심 목적:** 단기 매출 극대화. **Activation 40% 편중 허용** — 단, Brand Building 30% 최소 유지

## 특이 전략

| 항목 | 설정 |
|------|------|
| 60:40 | **30:70 (단기 2-4주)** — Activation 편중 허용 |
| 연간 총량 | 단 세일즈푸시가 연간 예산의 50% 초과 시 Redflag |
| Creative | 프로모션 메시지 강하게, BUT DBA 유지 |
| 플라이트 | Burst 2-4주 |
| 시즌 | 블프(11월)·크리스마스·설·추석·어린이날 |

## 채널 우선순위

1. 네이버 쇼핑·브랜드검색 (전환 극고)
2. 카카오톡 비즈보드·채널 (리타게팅·Open 3-5배)
3. 인플루언서 Micro/Nano (신뢰·전환 극고)
4. Meta·TikTok 퍼포먼스 광고
5. 이메일·SMS (기존 고객)

## KPI

| 축 | 지표 | 목표 |
|---|------|------|
| Activation | ROAS | 3.0+ (일반 2.0+) |
| Activation | CTR | 업계 평균 +50% |
| Activation | 전환율 | 업계 평균 +30% |
| Activation | 객단가 | 유지 또는 Δ + |
| Brand | Aided Awareness | 유지 (하락 금지) |
| Long-term | 재구매 전환 | 신규→재구매 10%+ |

## Gotchas

- 프로모션이 브랜드의 유일한 커뮤니케이션이 됨 → Commodity화
- 할인 중독 — 평상시 구매 감소, 할인 시기만 구매
- DBA 잃은 프로모션 크리에이티브 (가격만 노출)
- 연간 세일즈푸시 비중 >50% = Brand Building 실패 누적

## Redflag

- 연간 Brand Building 60% 미달 = Redflag 2 (Red)
- 프로모션 중 DBA 후퇴 = Redflag 4
- Last-click Attribution 단독 = Redflag 6
