# D6 — CSR·ESG (공익·ESG·퍼포스)

**핵심 목적:** 사회적 가치·ESG 기여를 브랜드 자산화. **진정성 없으면 Greenwashing 역풍**

## 특이 전략

| 항목 | 설정 |
|------|------|
| 60:40 | 70:30 (Brand 우선 — Activation 연계 지양) |
| 플라이트 | Continuous 12개월+ (단기 캠페인 = 진정성 실패) |
| 비즈니스 연관성 | 사업과 연결 필수 (무관한 기부 ≠ 브랜드 자산) |
| KOBACO 표준 | 공익광고 포맷 참조 |

## CSR·ESG 구조

| 유형 | 내용 | 예시 |
|------|------|------|
| E (Environment) | 기후·재생에너지·순환경제 | Patagonia·파타고니아 환경 기부 1% |
| S (Social) | 교육·인권·지역사회 | 유한킴벌리 "우리강산 푸르게" 40년+ |
| G (Governance) | 지배구조·투명성 | 지주·이사회 (→holdings-consulting) |
| 퍼포스 | 브랜드 근본 미션 | Dove Real Beauty·Always #LikeAGirl |

## 채널 우선순위

1. YouTube 장편 다큐멘터리 (감정·신뢰)
2. 네이티브·PR·기사 (정당성)
3. OOH (가시성·공공 기여 인식)
4. 자체 미디어 (웹사이트·Impact Report)
5. SNS 커뮤니티 (참여·공유)

## 진정성 체크리스트 (Greenwashing 방지)

- [ ] 사업과 연관성 (무관한 CSR ≠ 브랜드 자산)
- [ ] 측정 가능 목표 (수치·기간·검증기관)
- [ ] 3년+ 지속 계획
- [ ] Third-party 인증·파트너십 (환경단체·대학·NGO)
- [ ] Impact Report 정기 발행
- [ ] 임직원 참여 구조
- [ ] 고객·이해관계자 커뮤니케이션 루프

## KPI

| 축 | 지표 | 목표 |
|---|------|------|
| Brand | Purpose Association Δ | +10pt |
| Brand | Trust Index | 업계 Top 3 |
| Social Impact | 측정된 실질 기여 | 연간 보고 |
| Long-term | Employee NPS | 직원 자부심 Δ + |
| Long-term | Investor ESG Score | MSCI·Sustainalytics |

## 한국 사례

- **유한킴벌리 우리강산 푸르게푸르게** (1984~) — 40년+ 지속
- **오뚜기 착한 기업 이미지** — 일관된 세금·복지 투자
- **CJ 나눔문화** — 문화·교육 지원
- **SK 행복날개** — 사회적기업 지원

## Redflag

- 무관한 분야 일회성 기부 = Greenwashing 의심
- 측정 없는 CSR = 진정성 의심
- 3년 미만 단기 캠페인 = 효과 축적 불가
- Brand와 무관한 Purpose → 혼란·냉소
