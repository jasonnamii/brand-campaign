# D5 — 체험·팝업 (팝업스토어·콜라보·K-IP)

**핵심 목적:** 체류·공유·IP 활용으로 Fame 부스트. **한국 2024-2025 폭증 (2025년 3,077개·+109%)**

## 특이 전략

| 항목 | 설정 |
|------|------|
| 60:40 | 60:40 기본 (체험 자체가 Brand Building) |
| 목적 | 체류·SNS 공유·언론 노출 (Reach 증폭) |
| 기간 | 2-4주 기본·성수기 1주 |
| 장소 | 성수동 35%·연남·여의도·압구정·강남 |
| IP 콜라보 | K-pop·드라마·예능·웹툰·게임 |

## 팝업 운영 표준

| 항목 | 표준 |
|------|------|
| 일평균 방문 목표 | 500-3,000명 |
| 체류 시간 | 20-40분 |
| SNS 공유율 | 방문자의 30%+ |
| 언론 노출 | 3-5회 공중파·매체 기사 |
| 굿즈 판매 | 방문자의 20-40% 전환 |
| 비용 | 공간 2천만-1억 + 인테리어 3천-5억 + 운영 1천-3천 |

## AI 오감 마케팅 (2025 필수화)

1. **시각** — 몰입형 영상·LED·AR/VR
2. **청각** — 브랜드 사운드·징글·음악 큐레이션
3. **후각** — 브랜드 시그니처 향·공간향
4. **촉각** — 제품 직접 체험·인터랙션
5. **미각** — 시음·시식·카페 병설

## 채널 우선순위

1. 인플루언서 Micro/Nano (체험 리뷰·SNS 파급)
2. 네이버 플레이스·지도 (방문 유도)
3. 인스타그램 Reels (공유·해시태그)
4. 네이티브 기사 (모비인사이드·아웃스탠딩)
5. TikTok 챌린지 (확산)

## KPI

| 축 | 지표 | 목표 |
|---|------|------|
| Reach | 방문자 (기간 누적) | 5,000-100,000명 |
| Engagement | 체류·SNS 공유율 | 체류 20-40분·공유 30%+ |
| Fame | 언론 노출·해시태그 | 기사 3-5회·#브랜드 10K+ |
| Brand | Aided Awareness Δ | +5-10pt (기간 대비) |
| Activation | 온라인 매출 팝업 기간 | Δ +20%+ |

## Redflag

- 성수동만 고집 → 임대비 과다 (월 평당 50만원+)
- K-IP 의존도 과다 → IP 종료 후 브랜드 공백 (Redflag 8)
- AI 오감 2개 이하 = 2025 표준 미달
- IP 라이선스 계약 불명확 (→contract-reviewer)
- 체류 시간 <15분 = 기획 실패
