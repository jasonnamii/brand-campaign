## 체험·팝업 (한국 2024-2025)

- 2025년 팝업스토어 109% 증가 (3,077개)
- 성수동 35% 집중·연남동·여의도·압구정
- 평균 운영 2-4주·일평균 방문 500-3,000명
- AI 오감 마케팅 필수화 (시각·청각·후각·촉각·미각)
- 비용: 공간 2천-1억·인테리어 3천-5억·운영 1천-3천
- K-IP 콜라보 시 IP 라이선스·수익 배분 계약 필수

## Gotchas
- 영상 Hook 3초 무시 = TikTok·Reels·Shorts 알고리즘 실패
- 뒷광고 표시 누락 = Redflag 1 (Red)
- 팝업 공간 성수동 고집 → 임대비 과다. 연남·여의도·목적 분산 고려
- K-IP 콜라보 계약 없이 IP 사용 = 법적 리스크 (→contract-reviewer)
- 위기 72h 프로토콜 없이 즉흥 대응 = 2차 피해
- 캘린더 없이 론칭 = 제작 지연 → 플라이트 붕괴
